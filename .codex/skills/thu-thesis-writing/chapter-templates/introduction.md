# 引言章模板

用于第 1 章或独立的引言节。

## 推荐结构

```text
第1章 引言
1.1 研究概述
  1.1.1 研究背景
  1.1.2 关键问题
  1.1.3 研究意义
1.2 国内外研究现状
1.3 研究思路与主要工作
  1.3.1 研究思路
  1.3.2 主要工作
```

## 写作说明

### 1.1.1 研究背景

按以下顺序写作：

1. 宏观政策或行业背景
2. 具体工程矛盾
3. 现有运行/规划方法为何不足
4. 为何本文选题此时重要

起始模板：

```text
随着……，系统面临……挑战。
一方面，……；另一方面，……。
传统……方法通常着眼于……，但难以……。
因此，如何……成为……领域的重要问题。
```

### 1.1.2 关键问题

写明：

1. 一个论文层面的核心科学问题
2. 2 至 4 个子问题
3. 每个子问题 1 至 2 个技术难点

模板：

```text
基于上述背景，本文聚焦于"……"这一核心问题。
本文将该问题拆分为以下子问题：
1）……；
2）……；
3）……。

针对子问题 1，存在……和……两大技术难点。
针对子问题 2，存在……和……两大技术难点。
```

### 1.1.3 研究意义

将意义拆分为：

1. 建模或理论意义
2. 方法论意义
3. 应用或规划意义

### 1.2 国内外研究现状

按主题综述，不按作者年代罗列。

推荐模式：

1. 主题 A 研究现状
2. 主题 A 的小结
3. 主题 B 研究现状
4. 主题 B 的小结
5. 主题 C 研究现状
6. 主题 C 的小结
7. 尚存不足总结

小结模板：

```text
总体而言，当前研究仍存在以下问题：
1）……；
2）……；
3）……。
```

局部小结模板：

```text
总体而言，当前针对……的研究通常已经关注……，但在……、……和……方面仍存在不足。
```

### 1.3.1 研究思路

优先采用：

1. 先给出总体框架图
2. 问题导向
3. 研究框架
4. 最终决策支持或优化目标

### 1.3.2 主要工作

若论文路线简单，可将每章映射为一句话或一小段。

若路线较复杂，优先采用：

1. 先给出研究内容框架图
2. 按章节角色或研究阶段分组说明
3. 在分组说明中嵌入章节映射

简单模板：

```text
第2章……。
第3章……。
第4章……。
第5章……。
第6章……。
```

## 最终检查

- 由宽到窄的逻辑清晰
- 子问题与后续章节对齐
- 文献综述有缺口总结
- 章节安排与论文路线匹配
