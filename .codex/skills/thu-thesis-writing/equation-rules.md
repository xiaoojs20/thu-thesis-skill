# 公式、图与表规范

## 公式写作顺序（公式写作八股）

每条公式的呈现必须遵循以下强制性的"八股"顺序：

1. **引入句**：说明以下公式代表什么，以冒号（`：`）结尾。
2. **公式**：带 `\label` 的 `equation` 或 `align` 环境。
3. **叙述（显式命名）**：新段落/句子以 `式（\ref{label}）为 [模型/约束名称]` 开头。
   - 绝不能"空降"公式而不立即命名。
   - 即使使用 `\ref`，也要用中文全角括号 `（ ）`。
4. **变量说明**：用 `其中`（或长块用 `式中：`）解释符号。
5. **用途**：陈述建模意义或物理结果。

### 标签与显式引用规则

这是强制性要求，不是建议：

1. 每条编号显示公式必须有 `\label{eq:...}`。
2. 单个 `equation` 环境至少包含一个 `\label{eq:...}`。
3. 在 `align` 中，每条独立编号的公式行必须有各自的 `\label{eq:...}`。
4. 引入、解释或后续提及公式时，必须显式引用为 `式（\ref{eq:...}）`；不要仅依赖"上式""如下式"或"该公式"等措辞。
5. 对于公式组，用 `式（\ref{eq:a}）--式（\ref{eq:c}）` 或 `式（\ref{eq:a}）至式（\ref{eq:c}）` 引入，再解释每条公式的作用。
6. 若 `align` 块仅为一条公式的多行推导，则用 `\notag`/\nonumber` 抑制非独立行的编号，仅对需要引用的公式行加标签。

正确的 `align` 模式：

```tex
\begin{align}
    & P = \frac{U_s U_r}{X} \sin(\delta_s - \delta_r)
    \label{eq:P}
    \\
    & P_{\max} = \frac{U^2}{X}
    \label{eq:Pmax}
    \\
    & \Delta U = \frac{QX}{U}
    \label{eq:deltaU}
\end{align}

式（\ref{eq:P}）与式（\ref{eq:Pmax}）分别给出了……，式（\ref{eq:deltaU}）给出了……。
```

### 公式模式模板

```text
……构建了[模型名称]：
\begin{equation}
  [公式内容]
\label{eq:name}
\end{equation}
式（\ref{eq:name}）为[具体物理含义/约束类型]，其中 [变量1] 为……；[变量2] 表示……。
```


## `式中：` 规则

1. 以 `式中：` 开头。
2. 按变量出现顺序排列。
3. 用 `；` 分隔各项。
4. 先解释物理含义，再解释角色、范围或上下界。

模板：

```text
式中：X 表示……；Y 表示……；Z 为……，其上限为……。
```

## 公式后还需说明什么

不要停在符号定义。还要说明建模功能：

- 描述不确定性
- 保持时序相关性
- 量化季节波动
- 定义平衡约束
- 降低计算复杂度
- 支撑后续场景分析

## 图规范

### 功能分工

图用于：

- 背景认知
- 研究框架
- 拓扑结构
- 流程
- 动态过程
- 平衡曲线
- 灵敏度分析

### 标题风格

标题保持名词性，不用结论性表述。

良好示例：

- `某系统拓扑结构`
- `某方法框架`
- `某结果对比分析`
- `某月平衡结果`
- `某灵敏度分析`

不要把最终结论放进标题。

### 正文中的图解读

采用以下模式：

```text
图 x 展示了……。
由图可知，……。
相较于方案 A，方案 B ……。
```

## 表规范

表用于：

1. 方案描述
2. 参数设置
3. 结构化结果对比
4. 文献汇总

标题同样保持名词性：

- `对比方案描述`
- `关键参数设置`
- `不同方案的优化结果对比分析`

## 图与表的分工

- 表承载结构化的数值对比
- 图承载趋势、拓扑、过程和机理

若紧凑的表能更好表达结果，则不要用图。

## 引用风格

工科论文正文优先采用数字引用：

- `[1]`
- `[2]`
- `[9-10]`

引用放在所支撑的技术论断之后。

## LaTeX 约定

若输出 LaTeX 兼容文本，默认采用：

- 单行公式 -> `equation`
- 多行公式 -> `align`
- 正文中的公式引用 -> `式（\ref{eq:...}）`
- 每条编号公式必须有 `\label{eq:...}`
- `align` 中每条独立编号行必须有各自的 `\label{eq:...}`
- 图 -> `figure`
- 表 -> `table`
- 标签：
  - `eq:...`
  - `fig:...`
  - `tab:...`
