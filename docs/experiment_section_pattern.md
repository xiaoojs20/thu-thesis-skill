# 算例分析章节模式

> 提炼对象：PDF 中第 2 章到第 5 章的 `算例分析`。
>
> 目标：给未来论文写作提供可直接套用的“实验/算例章节骨架”。

## 1. 总体观察

四个核心章节都包含独立的 `算例分析`，并且不是简单的“实验设置 + 结果”两段式，而是更完整的验证链：

1. 说明验证目标
2. 交代数据来源和计算环境
3. 定义测试系统或方案组
4. 给出参数/边界条件
5. 展示对比结果
6. 从图表中解释原因
7. 用一句话回收“证明了什么”

## 2. 各章算例结构

## 2.1 第 2 章算例结构

`2.5 算例分析` 的子结构是：

1. `2.5.1 风光时间序列季节性分解`
2. `2.5.2 气候分量模拟结果`
3. `2.5.3 季节分量模拟结果`
4. `2.5.4 日间分量模拟结果`
5. `2.5.5 重构时间序列对比分析`

特点：

- 先验证中间模块，再验证整体重构效果。
- 不是直接给最终结果，而是按“分量级验证 -> 系统级验证”展开。
- 最后一个小节专门做综合对比，完成收束。

这类写法适合“提出了新建模方法”的章节。

## 2.2 第 3 章算例结构

`3.5 算例分析` 的子结构是：

1. 先给 `4 组方案对比`
2. 再按算例对象分成两个系统：
   - `3.5.1 Garver-6 节点系统测试算例`
   - `3.5.2 中国西北电力系统测试算例`
3. 每个系统内部再写：
   - `边界条件说明`
   - `优化结果对比分析`

特点：

- 先把方案定义清楚，再进入结果。
- 一个小算例验证机制正确性，一个实际系统验证可扩展性。

这类写法适合“提出了新求解框架/新模拟方法”的章节。

## 2.3 第 4 章算例结构

`4.5 算例分析` 的子结构是：

1. 先说明 4 组方案设置
2. 再分两个系统：
   - `4.5.1 IEEE-5 节点系统测试算例`
   - `4.5.2 中国西北电力系统测试算例`
3. 第一个系统内部继续分：
   - 电氢耦合算例
   - 电热耦合算例

特点：

- 先做机制验证，再做复杂系统验证。
- 结果不是只比较总成本，而是分系统、分能源链解释。

这类写法适合“提出了协同优化规划模型”的章节。

## 2.4 第 5 章算例结构

`5.5 算例分析` 的子结构是：

1. 先给算例设置表
2. 再分两个系统：
   - `5.5.1 Garver-6 节点系统测试算例`
   - `5.5.2 IEEE-RTS79 系统测试算例`
3. 在 Garver-6 算例中增加灵敏度分析

特点：

- 先用方案组合验证风险项是否起作用
- 再用灵敏度分析证明参数变化规律
- 最后在更大系统上验证模型有效性

这类写法适合“引入风险/鲁棒性/机会约束”的章节。

## 3. 统一的算例章节模板

综合第 2 到第 5 章，可提炼出一套稳定模板：

### 模板 A：方法验证型算例

1. 算例目标
2. 数据来源与计算环境
3. 分模块结果
4. 整体结果重构/对比
5. 结论

适用：新模型、新指标、新分解方法。

### 模板 B：方案对比型算例

1. 方案设置
2. 测试系统说明
3. 参数设置
4. 结果对比
5. 结果解释
6. 结论

适用：新求解方法、新规划模型、新优化框架。

### 模板 C：风险分析型算例

1. 方案设置
2. 场景年或不确定性设定
3. 基础结果
4. 灵敏度分析
5. 更大规模系统验证
6. 结论

适用：风险评估、鲁棒优化、概率约束。

## 4. 算例段落的微观写法

## 4.1 算例开头怎么写

每个算例节的开头基本都回答 4 个问题：

1. 用什么系统
2. 为什么选这个系统
3. 比较哪些方案
4. 在什么环境下计算

推荐模板：

```text
本节基于……系统进行算例分析，以验证……。
共考虑……组方案，对比设置如表 x 所示。
所有算例通过……实现，并在……环境下测试。
```

## 4.2 结果段怎么写

结果段几乎都遵循“图表引用 -> 观察 -> 对比 -> 结论”的四步。

推荐模板：

```text
表 x 对比了方案 A 和方案 B 的规划结果。
由表可知，方案 B 在……方面优于方案 A，主要体现在……。
相应地，……由……降低至……。
通过上述对比，有效证明了……。
```

## 4.3 图像结果怎么解释

图像结果解释不是重复图题，而是解释变化规律。

常见解释角度：

1. 峰谷出现在什么季节/月
2. 充能/放能分别发生在什么时段
3. 方案 A 与方案 B 的差异来自什么机制
4. 结果是否体现出预期的季节性规律

例如该论文反复使用：

- `体现出明显的季节波动特性`
- `表现为“两充两放”`
- `在……季节开始出现储能放能`
- `有效证明了本文方法的有效性`

## 5. 图表搭配规则

算例章节中，图和表分工非常明确：

1. 表格负责：方案、参数、总量结果
2. 图负责：拓扑、过程、平衡曲线、敏感性

具体搭配方式：

- 先用表给方案设置
- 再用图给系统拓扑
- 再用表给关键参数
- 最后用图解释动态结果

这是工科算例最稳妥的组织方式。
