# 学位论文结构分析

> 适用对象：中文工科博士论文，尤其是“问题提出 -> 方法建立 -> 模型求解 -> 优化配置 -> 风险评估”型论文。
>
> 依据来源：
> - 成品论文 PDF：`thuthesis-jiang_haiyang.pdf`
> - LaTeX 项目：`thuthesis-v7.5.0`

## 1. 整体结构

该论文不是“多篇文章拼接式”结构，而是严格的“1 个总问题 + 4 个核心技术章节 + 1 个总结章节”结构。

整体顺序固定为：

1. 封面、委员会、授权说明
2. 中文摘要
3. 英文摘要
4. 目录、插图清单、附表清单
5. 符号和缩略语说明
6. 第 1 章 引言
7. 第 2 章 方法基础 1：对象不确定性分析与建模
8. 第 3 章 方法基础 2：运行模拟/求解加速方法
9. 第 4 章 规划模型：核心资源建模与优化配置
10. 第 5 章 风险评估：风险量化与考虑风险的优化配置
11. 第 6 章 结论
12. 参考文献、致谢、声明、简历等后置材料

## 2. 正文章节主骨架

正文主线不是按“数据、方法、实验、讨论”重复展开，而是按研究链条逐章推进：

1. 第 1 章：提出问题，定义研究对象，说明文献缺口，给出研究框架。
2. 第 2 章：先把“季节不确定性”这个研究对象描述清楚，并完成基础建模。
3. 第 3 章：解决“全年时序 + 季节储能”导致的求解复杂度问题。
4. 第 4 章：建立考虑季节性储能的优化规划模型。
5. 第 5 章：在规划模型上进一步加入风险量化与风险约束。
6. 第 6 章：按 4 项核心成果回收全文贡献。

这类结构适合未来写作时采用：

- 第 2 章回答“对象怎么刻画”
- 第 3 章回答“模型怎么算”
- 第 4 章回答“资源怎么配”
- 第 5 章回答“风险怎么控”

## 3. 各章结构模式

### 3.1 引言章固定模式

第 1 章采用三段式：

1. `1.1 论文研究概述`
   - `1.1.1 研究背景`
   - `1.1.2 关键问题`
   - `1.1.3 研究意义`
2. `1.2 国内外研究现状`
   - 按主题拆分综述，而不是按作者/年份流水账
3. `1.3 本文研究思路与主要工作`
   - `1.3.1 研究思路`
   - `1.3.2 主要工作`

这个模式说明：引言章内部已经完整回答了“为什么做、别人做到哪、本文怎么做”。

### 3.2 方法/模型章节固定模式

第 2 章到第 5 章高度一致，基本都遵循下列模板：

1. `x.1 概述`
2. `x.2` 定义/符号表/问题分析
3. `x.3` 方法或模型主体
4. `x.4` 进一步展开的模型、框架或约束
5. `x.5 算例分析`
6. `x.6 本章小结`

具体映射如下：

| 章节 | 结构骨架 | 作用 |
| --- | --- | --- |
| 第 2 章 | 概述 -> 不确定性分析 -> 建模方法 -> 各分量建模 -> 算例 -> 小结 | 先分析对象，再建模 |
| 第 3 章 | 概述 -> 符号表 -> 运行模拟模型 -> 加速框架 -> 算例 -> 小结 | 先定义变量，再给求解框架 |
| 第 4 章 | 概述 -> 符号表 -> 季节性储能模型 -> 优化规划模型 -> 算例 -> 小结 | 先建储能模型，再嵌入规划 |
| 第 5 章 | 概述 -> 符号表 -> 风险评估模型 -> 考虑风险的优化配置 -> 算例 -> 小结 | 先量化风险，再把风险并入规划 |

### 3.3 典型小节层级

三级标题的组织也很稳定，常见模式有三类：

1. `定义/分类/指标`
   - 例如：定义、场景、指标体系、挑战
2. `模型分解`
   - 例如：目标函数、运行约束、平衡约束、子模型
3. `算例子任务`
   - 例如：边界条件说明、结果对比、灵敏度分析

未来写作时，如果一个二级节承担“模型建立”功能，三级标题优先按下面顺序排：

1. 目标或问题定义
2. 变量与参数
3. 目标函数
4. 核心约束
5. 模型框架或求解流程

| 章节 | PDF 页数 | 中文字符数估计 | 正文占比 |

| 第 6 章 结论 | 4 | 3090 | 4.0% |

可以直接提炼出一个写作配比：

- 引言是最长章，约占正文 30%。
- 每个核心技术章占 15% 到 20%。
- 结论明显短于技术章，控制在正文 4% 到 6%。

## 5. 图表密度分布

按插图清单和附表清单统计：

| 章节 | 图数 | 表数 | 图表特点 |
| --- | ---: | ---: | --- |
| 第 1 章 | 7 | 4 | 背景图、框架图、综述对比表 |
| 第 2 章 | 20 | 8 | 方法流程图、统计图、建模参数表最多 |
| 第 3 章 | 14 | 8 | 框架图、拓扑图、方案对比图表 |
| 第 4 章 | 17 | 4 | 系统拓扑图、月平衡图、方案结果图 |
| 第 5 章 | 13 | 3 | 风险图、灵敏度图、方案设置表 |

这说明：

1. 引言图表只负责“建立问题”和“展示框架”，不负责堆结果。
2. 方法章图表最密，用于解释流程、变量关系、统计规律。
3. 实验章和规划章图多于表，强调结果可视化。
4. 表格主要承担“参数、方案、结果对比”三种功能。

    1.2.2 主题B研究现状
    1.2.3 主题C研究现状
  1.3 研究思路与主要工作
    1.3.1 研究思路
    1.3.2 主要工作

  2.1 概述
  2.2 对象定义与指标

  2.5 算例分析

  3.1 概述
